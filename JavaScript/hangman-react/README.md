# Hangman

Hangman implementation in React.

## Demo

[Check out the demo](https://hangman-react-js.netlify.app/)

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm run start

# Build for production
npm run build

```
