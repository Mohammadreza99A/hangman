# Hangman

Hangman implementation in VueJs.

## Demo

[Check out the demo](https://hangman-vue-js.netlify.app/)

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:8080
npm run serve

# Build for production
npm run build

# Lint and fix files
npm run lint
```
